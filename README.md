# JavaScript Mini Projects
A collection of fun and practical mini projects built using HTML, CSS, and JavaScript. These projects helped me improve my frontend development skills by working with DOM manipulation, event handling, APIs, responsive design, and JavaScript logic.
## Projects Included
### Tic Tac Toe
A two-player game where players take turns placing X and O on a 3×3 grid. The game automatically detects winners and draws.
**Features:**
- Player turn indicator
- Winner detection
- Draw detection
- Reset and New Game options
